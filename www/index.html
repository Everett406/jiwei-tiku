关于积微

积微，取自"积累微小"。我们相信，每一次微小的努力，都会在某个时刻连成一条通往远方的路。

这是一个简单纯粹的刷题工具，支持导入 JSON 和 Excel 格式的题库。无论你是备考证书、复习课程，还是日常积累，都可以把题目带在身边，随时做上几道。

如果你发现题目有误，或者有什么想加的功能，可以点页面上的「反馈」告诉我。

🤖 AI 辅助导入

如果你有 PDF、Word、Excel 或截图格式的题库素材，把下面这段提示词复制给 AI（Kimi、ChatGPT、豆包、Claude 等），AI 会帮你整理成可导入的标准格式。

你是一位专业的题库整理助手。我会发给你一份题库素材（可能是 PDF、Word、Excel、图片或截图），请你帮我把它转换成标准格式，方便导入刷题软件。

目前只支持单选题和判断题。如果素材里有填空题或多选题，请跳过不处理。

请任选以下一种输出格式：

【格式一：JSON】
返回一个标准 JSON 数组，每个题目包含：
- id：题号（从 1 开始递增）
- question：题干文本（不要带题号前缀）
- answer：正确答案字母，必须大写（如 A、B、C）
- options：选项数组，每项包含 key（A/B/C...）和 text（选项内容）

示例结构：
[
  {
    "id": 1,
    "question": "人为差错主要是指？",
    "answer": "A",
    "options": [
      {"key": "A", "text": "人的行为结果超出了可接受的界限"},
      {"key": "B", "text": "人所犯的任何错误"},
      {"key": "C", "text": "有的人故意为之"}
    ]
  }
]

注意：
1. 判断题请统一用两个选项：A=正确/对/是，B=错误/错/否。
2. 确保 JSON 格式合法，不要带 markdown 代码块标记，直接输出纯 JSON 文本。
3. 如果题干里有无法识别的图片或表格，请在对应位置标注【图片】。

【格式二：填充好的 Excel】
如果你直接给我填充好的 .xlsx 文件也可以。请使用以下 7 列顺序：
第 1 列：题干
第 2 列：选项 A
第 3 列：选项 B
第 4 列：选项 C
第 5 列：选项 D
第 6 列：选项 E（没有则留空）
第 7 列：答案（填字母，如 A、B）
📋 Excel 模板下载

也可以下载空白模板，手动填写后导入。请严格使用以下 7 列格式：

1题干
2选项 A
3选项 B
4选项 C
5选项 D
6选项 E
7答案

判断题只需填 A/B 两列，答案填"正确/错误"或"A/B"；单选题按实际选项数填写即可。
